# Digital Addiction Pattern Analyser

A comprehensive data analytics and Streamlit dashboard designed to track, visualize, and analyze screen time and productivity metrics to understand digital addiction patterns.

##  Features
* **Interactive Dashboard:** Built using Python and Streamlit for real-time data insights.
* **Data Cleaning:** Processed and filtered a complex student performance dataset down to 15 completely valid and clean user profiles for accurate analysis.
* **Behavioral Insights:** Tracks screen time habits, app usage patterns, and productivity drops.

##  Tech Stack
* **Language:** Python
* **Libraries:** Streamlit, Pandas, Plotly / Matplotlib
* **Data Source:** Cleaned Student Performance Dataset
